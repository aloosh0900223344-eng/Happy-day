<h1 align="center">
  🎉 Happy Day - تطبيق هدية عيد الميلاد 🎂
</h1>

تطبيق Flutter لإنشاء فيديوهات هدايا عيد ميلاد جميلة من الصور!

## ✨ المميزات

- 🎨 رسالة عيد ميلاد جميلة وديناميكية
- 📸 اختيار الصور من الاستديو (حتى 10 صور)
- 🎬 تحويل الصور لفيديو احترافي
- ⏱️ التحكم في مدة كل صورة
- 🎪 أنواع انتقالات مختلفة
- 📤 مشاركة الفيديو بسهولة

## 🚀 البدء السريع

### المتطلبات
- Flutter SDK 3.0+
- Android SDK
- Visual Studio Code أو Android Studio

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/aloosh0900223344-eng/Happy-day.git
cd Happy-day

# تثبيت المعتمدات
flutter pub get

# تشغيل التطبيق
flutter run
```

## 📁 هيكل المشروع

```
lib/
├── main.dart                    # نقطة البداية
├── screens/
│   ├── birthday_message_screen.dart    # شاشة الرسالة
│   ├── photo_picker_screen.dart        # شاشة اختيار الصور
│   └── video_creator_screen.dart       # شاشة إنشاء الفيديو
└── utils/                       # دوال مساعدة
```

## 📦 المعتمدات الرئيسية

- **image_picker**: اختيار الصور من الاستديو
- **video_player**: تشغيل الفيديوهات
- **ffmpeg_kit_flutter**: معالجة الفيديو
- **share_plus**: مشاركة الفيديوهات
- **path_provider**: إدارة المسارات

## 📱 الشاشات

### 1. شاشة الرسالة (Birthday Message)
- عرض رسالة عيد ميلاد جميلة
- زر للانتقال لاختيار الصور

### 2. شاشة اختيار الصور (Photo Picker)
- استعراض وإضافة الصور
- عرض الصور المختارة بشكل Grid
- إمكانية حذف الصور

### 3. شاشة إنشاء الفيديو (Video Creator)
- التحكم في مدة كل صورة
- اختيار نوع الانتقال
- إنشاء الفيديو والمشاركة
